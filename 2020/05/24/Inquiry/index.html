고객 문의 정리 및 알림

회사에 문의 내용이 접수 되었을 시 접수 내용을 따로 문서에 정리하고 그 문서를 공유하느라 이메일을 발송하고 내용이 업데이트 될 때마다 또 보고하고 내용을 정리하는 상황입니다.

Share Point List를 활용하여 자동입력 및 모든 문의내용을 한곳으로 모이게 하고 바로 알림을 보내어 내용을 공유해보도록 합니다.

홈페이지의 문의 접수란의 화면입니다.

문의 접수 느낌이 나도록 MS Forms를 이용하여 내용입력 화면을 만들었고 Forms 접속 URL 주소를 Wordpress 안에 넣어놓았습니다.

고객이 내용 작성 후 ‘제출’을 클릭하면 이 내용을 자동으로 입력되게 할 것입니다.

우선 문의 접수를 처리하는 해당 MS Teams의 Share Point에 새로운 List를 생성합니다.

접수받는 Forms와 연동될 수 있게 동일한 내용으로 만들어줍니다.

고객과의 진행 상황을 등록할 수 있게 ‘내용’, ‘진행상황’ 등록란을 추가합니다.

Share Point보다는 늘 접속해 있는 Teams에서 업무를 처리하기 위해 상단 탭에 연동을 합니다.
‘+’ 을 클릭하여

Share Point를 선택합니다.

‘Lists’를 선택하면 Share Point에 만들어 놓은 리스트들이 나오고 사용할 리스트를 선택하면

이렇게 Teams 내에 Share Point List가 자동으로 연동되게 됩니다.

Power Automate

이제 고객이 MS Forms에 문의 내용을 입력 후 ‘제출’을 하면 Share Point List에 자동으로 입력될 수 있게 Power Automate를 사용하여 자동화 흐름을 만듭니다.

 Power Automate : 프로세스 자동화 설계 https://flow.microsoft.com/

왼쪽 My flows를 선택 ‘+New’ -> ‘Automated’를 선택합니다.

자동으로 실행 할 ‘Trigger’로 MS Forms를 사용할 것 이므로 첫번째 Forms를 선택합니다.

어떤 내용의 Forms를 사용할 것 인지 정하기 위해 Forms id를 입력합니다.

본인이 만든 Forms경우 제목이 나타납니다.
그러나 다른 Forms를 가져와야 할 경우

https://forms.office.com/Pages/DesignPage.aspx?origin=OfficeDotCom&lang=en-US#FormId=RUbEfkw2ZEOC8KVVY743vUdyqq-mxexDgseZYSIL7iVUMVZCWElPQzE0SVdVUUk2WjRKRkxZVVZWOCQlQCN0PWcu&Preview=%7B%22PreviousTopView%22%3A%22None%22%7D&TopView=Preview

Forms URL 전체 주소에서

이 부분을 입력해주면 됩니다.

입력 된 Forms 내용을 갖고 오게하는

‘Get response details’를 선택하고 내용을 입력합니다.

가져온 내용을 입력하기 위해

SharePoint 의 ‘Create Item’을 선택하고 ‘Site Adress’에 해당되는 SharePoint 위치
‘List Name’ 및 입력해야 될 각 내용을 설정합니다.

Share Point List에 내용이 입력되면 Teams의 게시물에 자동으로 내용이 입력되게
‘Post a message’를 설정합니다.

담당자가 문의가 온 것을 바로 알수 있게 이메일 발송을 설정합니다.

전체 자동화 흐름의 모습입니다.

내용 변경 시 알림 설정

문의 내용에 변동 사항이 생길 시 알림을 보내는 자동화 흐름을 만듭니다.

위와 동일하게 새로운 자동화 흐름만들기를 선택합니다.

이번에는 자동화 실행 조건을 ‘Share Point의 ‘When an item is created or modified’로 설정합니다.

‘When an item is created or modified’에서 알림을 받을 Share Point List를 설정합니다.

이메일로 담당자에게 알림을 발송합니다.

어떤 문의 내용이 변경됐는지 알려주기 위해 메일 내용란에 ‘Title’을 넣어 줍니다.

이제 홈페이지에서 고객이 문의를 제출하게 되면 쉐어포인트에 내용이 등록되고
담당자에게 바로 이메일이 발송되어 확인 후 팀즈에서 확인 및 진행상황 정리까지 자동으로 이루어지게 됩니다.
